# KC
Plusiseurs classes sont disponibles dans ce projet, ayant chacune des rôles précis  : 
- Graphe ( Implémentation générale du graphe et sa caractérisation )  
- Noeud ( Caractérisation d'un noeud )
- Lien ( Caractérisation d'une arête )
- PCC ( Implémentation des algorithmes de Plus Court Chemins pour les graphes orientés , non-orientés ou même pondéré .
- Coloration ( Coloration d'un graphe )
- Dessin ( Visualisation d'un graphe )
  Dans la classe Graphe , on y retrouve plusiseurs methodes ainsi que son contructeur naturel :
  - Orientation
  - Ponderation
  - Matrice ( Creation de sa matrice d'adjacence )
  - Mat_Incidence ( Creation de sa matrice d'incidence )
  - Liste_Succ ( Creation d'une Liste de Succeseurs )
  - Liste_Pred ( Creation d'une Liste de Predecesseurs )
  - DFS ( Algorithme de Parcours en Profondeur ou Deep First Search)
  - BFS ( Algorithme de Parcours en Largeur ou Breath First Search)
  - Nombre de Chemins
  - Existence d'un Chemin et sa recherche 
  - Deg_Max ( Degre du graphe )
  - Connexité
  - Composantes Connexe
  - Existence d'un Circuit et sa recherche
  - Listes des Niveaux,Degres,Dates(date de decouverte et date de fin)
  - Méthodes d'Affichage
  - Méthodes de Tri (Tri Croissant et Tri Decroissant)

Dans la classe PCC , on y retrouve plusiseurs methodes ainsi que son contructeur naturel :
 - Trois Méthodes de PCC :
 - Dijsktra
 - Bellman-Ford
 - Floyd-Warshall-Roy
Dans la classe Coloration , on y retrouve l'Algorithme de Welsh-Powell ainsi que son contructeur naturel
Dans la classe Desin , on y retrouve plusiseurs methodes ainsi que son contructeur naturel :
- Visualiation d'un graphe (munie d'une disposition aléatoire des points )
- Visualisation d'un graphe orienté ( munie d'une disposition hierarchique grace à l'Algorithme de Sugiyama)
